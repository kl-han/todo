Calendar Presentation
=====================

The calendar is a read model over:

* Task start dates.
* Task due dates.
* Recurring occurrences.
* Daily plan time blocks.
* Completed focus sessions.

Views
-----

* Agenda.
* Day.
* Week.
* Month.

Recommended implementation order:

1. Agenda list.
2. Week view.
3. Day timeline.
4. Month overview.

Month view is visually attractive but usually less useful for execution
than the agenda and week views.
