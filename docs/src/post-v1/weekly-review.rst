Weekly Review Report
====================

Sections
--------

.. list-table::
   :header-rows: 1
   :widths: 35 65

   * - Section
     - Calculation
   * - Completed tasks
     - Count by quadrant and tag
   * - Carryover
     - Planned but incomplete tasks

   * - Application usage

   * - Distraction time

   * - Delegated follow-up

   * - Cleanup candidates
     - Stale Q4 tasks

Facts versus interpretations
----------------------------

The report distinguishes facts from interpretations:

.. code-block:: text

   Fact: 6h 20m was spent in development tools.
   Fact: 2h 10m was spent in entertainment applications.
   Inference: Q2 work received less time than planned.
   Recommendation: Move one Q2 task into Monday's daily plan.

Do not automatically judge an application as productive. Let the user
map applications to categories, and optionally map categories to tags
or projects.
