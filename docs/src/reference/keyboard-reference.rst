Keyboard Reference
==================

.. list-table::
   :header-rows: 1
   :widths: 25 45 30

   * - Keys
     - Action
     - Since
   * - ``Alt+1``
     - Matrix tab
     - 0.3
   * - ``Alt+2``
     - Tasks tab
     - 0.3
   * - ``Alt+3``
     - Tags tab
     - 0.3
   * - ``h`` / ``j`` / ``k`` / ``l``
     - Focus left / down / up / right (outside text input)
     - 0.3
   * - ``Enter``
     - Toggle focused task's completion; submit in text fields
     - 0.3
   * - ``Enter``
     - Confirm the highlighted autocomplete suggestion in title editing
     - 3.0
   * - ``Up`` / ``Down``
     - Move through autocomplete suggestions in title editing
     - 3.0
   * - ``Ctrl+P`` / ``Ctrl+N``
     - Move to the previous / next autocomplete suggestion in title editing
     - 3.0
   * - ``Tab`` / ``Shift+Tab``
     - Focus traversal
     - 0.3

iOS and touch web rely primarily on touch; hardware-keyboard support
inherits this table.
